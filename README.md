# Demo Docs

Przykładowy projekt dokumentacji stworzonej przy użyciu MkDocs + Material for MkDocs.

## Jak uruchomić

1. Zainstaluj MkDocs:

```
pip install mkdocs-material
```

2. Uruchom lokalny serwer:

```
mkdocs serve
```

3. Wejdź na `http://127.0.0.1:8000`

## Publikacja na GitHub Pages

```
mkdocs gh-deploy
```
